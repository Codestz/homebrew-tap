# Homebrew Tap

Custom Homebrew formulae by [codestz](https://github.com/codestz).

## Install

```bash
brew tap codestz/tap
brew install <formula>
```

## Formulae

| Formula | Description |
|---------|-------------|
| `claude-hindsight` | 20/20 hindsight for your Claude Code sessions |
| `krait` | Code intelligence CLI for AI agents — LSP-backed symbol search and semantic editing |
| `mcpx` | MCP servers as CLI tools — built for AI agents |
